# Changelog

All notable changes to this project will be documented in this file.

## v0.2-preview

This is the first public preview of **Memory With Rooms**.

### Included

- a reusable layered-memory skill for OpenClaw agents
- a sanitized demo workspace
- adoption and architecture references
- scripts for layout checks, registry validation, registry generation, and smoke testing
- guidance for using curated rooms with dreaming and palace/wiki layers

### Design goals

- separate drafts from curated recall
- preserve process without letting it dominate retrieval
- make relationships, preferences, and open questions easy to revisit
- support gradual adoption in existing workspaces

### Notes

- this preview is intentionally conservative
- it focuses on memory structure and workflow, not a full automatic plugin pipeline
- all personal content from the source system was removed or sanitized before publication
